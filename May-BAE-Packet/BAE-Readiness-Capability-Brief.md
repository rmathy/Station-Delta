# Station Delta — 1-Page Capability Brief (BAE Alignment)

## Company
**Station Delta**  
Founder: Richard Mathy  
Product: Scenario-based command-line readiness simulation platform

## What We Do
Station Delta delivers mission-style technical training simulations that build practical operator fluency, not passive content recall. Learners complete objective-based command tasks with real-time coaching, assessment pressure, and measurable performance telemetry.

## Current Product State
- Public build live: `https://station-delta-one.vercel.app/`
- Core sims active: Vibe Cadet, Vibe Engineer
- 13-course architecture planned for role/track expansion
- Analytics capture active for step completion, retries, hints, and time-to-resolution

## Problem We Solve
Technical teams are often evaluated on outputs without consistent evidence of skill readiness. Existing learning workflows are frequently:
- too passive,
- too generic,
- weak on applied performance measurement.

Station Delta converts training into repeatable, measurable command performance under guided pressure.

## Alignment to BAE Interest Areas
### 1) Automation in constrained/air-gapped contexts
- Simulation-first workflows can be packaged for restricted environments where external dependencies are limited.
- Objective engine enables repeatable verification of operator procedures.

### 2) AI for vulnerability/threat response behavior
- Learners are trained on detection/response command habits and diagnostic decision paths.
- Progress telemetry identifies weak response patterns for remediation.

### 3) Multi-objective analytics for likely-success TTPs
- Event stream supports analysis of which tactic paths produce fastest correct outcomes.
- Retry/hint/timing data enables probabilistic readiness mapping by competency.

### 4) Modeling and simulation for maturity acceleration
- Structured scenario progression from beginner through operational complexity.
- Faster iteration loops than static courseware; updates can ship rapidly based on observed learner friction.

## Differentiators
- Strong instructional character model (LOLA coaching, JANET assessment pressure)
- High engagement with measurable command-level outcomes
- Fast content iteration pipeline and deploy cadence
- Built by an educator with deep pedagogy focus and AI-enabled engineering workflow

## Pilot Proposal (Initial)
- Duration: 4–6 weeks
- Scope: 1 role track, 10–30 learners
- Deliverables:
  - baseline and final readiness snapshots,
  - friction map by objective,
  - recommendations for scenario hardening.
